A self-contained browser application for encoding and decoding the Ogham script. Part of the [[🌿 Celtic Filid & Ogham]] project.

| File | Description |

## Features

- **Latin → Ogham** encoding with digraph support (NG, ST)
- **Ogham → Latin** decoding
- **Forfeda** (supplementary letters): EA, OI, UI, IO, AE, P
- **Multi-stem** rendering — use `.` to separate inscriptions on different stems
- **Vertical mode** — traditional bottom-to-top inscription layout
- **Feather marks** ᚛ ᚜ — authentic inscription delimiters
- **Spacing / Druim lines** toggle
- **Fibonacci Reckoning** — positional Fibonacci sum, feda count, harmonic weight
- **Feda Breakdown** — per-letter name, tree, Aicme, and Fibonacci index
- **Clickable Aicme reference panels** — insert glyphs directly
- Copy-to-clipboard for both Ogham and Latin output

## How to Use

1. Open `Ogham Calculator.html` in a browser (double-click or drag to browser)
2. Type Latin text in the input field → Ogham appears on the stone
3. Use `.` to separate multiple inscription stems
4. Click any glyph in the left panel to insert it (switches to decode mode)
5. Paste Ogham script to decode → Latin reading appears below
